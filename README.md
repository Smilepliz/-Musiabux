# Musiabux

## Как запустить (проще всего)

**Скачайте готовый exe и запустите — ничего ставить не нужно.**

1. Откройте [Actions](https://github.com/Smilepliz/-Musiabux/actions).
2. Выберите последний зелёный запуск **Build EXE**.
3. Внизу страницы нажмите **Artifacts** → скачайте **Musiabux-exe** (zip с exe внутри).
4. Распакуйте и запустите **Musiabux 0.0.0.exe** (или аналогичный по имени).

Готово — приложение откроется в своём окне.

---

Если хотите запускать из исходников (нужен [Node.js](https://nodejs.org)): установите Node.js, затем дважды нажмите **Запуск.bat**.
