# Musiabux

## Как запустить

### Вариант 1 — один клик (если установлен Node.js)
Дважды нажмите **`Запуск.bat`**. Откроется браузер с приложением. Окно с сервером не закрывайте.

[Node.js](https://nodejs.org) нужен только один раз; при первом запуске скрипт сам установит зависимости.

### Вариант 2 — без установки Node.js
Скачайте готовый **Musiabux.exe** в разделе [Releases](https://github.com/Smilepliz/-Musiabux/releases) и запустите его. Ничего дополнительно ставить не нужно.

*(EXE появляется в Releases после того, как создан тег версии, например `v1.0.0`.)*
